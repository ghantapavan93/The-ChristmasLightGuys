# The Christmas Light Guys — Website

Premium marketing website for **The Christmas Light Guys**, a locally owned,
bonded &amp; insured **residential & commercial holiday-lighting company** serving
Austin and Central Texas.

> *You enjoy the season. We handle the ladders.*

## ✨ Highlights

- **Cinematic homepage** — hero image slideshow with a "power-on" light surge,
  a draped café-light garland, a flashing "Shine This Christmas" headline,
  parallax depth, and drifting snow.
- **Design Your Glow** — an interactive house where tapping areas (roofline,
  trees, walkways…) lights them up and carries the plan into the estimate.
- **Guided estimate form** — a 5-step quote flow with real lead delivery,
  photo uploads, and a celebratory confirmation.
- **Filterable gallery** with a full-screen lightbox and "request this look."
- **Dedicated Residential & Commercial pages** (SEO-optimized).
- **FAQ, reviews, About, service area**, and a sticky mobile call/estimate bar.
- Fully **responsive** and **reduced-motion friendly**.

## 🗂 Structure

```
site/
  index.html            # Homepage
  residential.html      # Residential services page
  commercial.html       # Commercial services page
  assets/
    css/style.css        # Design system + all styling
    js/main.js           # All interactions (vanilla JS, no framework)
    img/                 # Photography + hero slideshow (img/hero/hero-1..7.jpg)
Pictures to Use/         # Original source photography
```

## 🚀 Run locally

It's a static site — no build step. Either:

- **Double-click** `site/index.html`, or
- Serve it: `npx serve site` then open the shown URL.

## ☁️ Deploy (Vercel)

The site lives in the `site/` folder, so set the project **Root Directory** to
`site` (Framework preset: **Other**). No build command needed.
